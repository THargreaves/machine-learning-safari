Machine Learning Safari
=======================

   :hidden:
   :maxdepth: 1

Machine Learning Safari is an esoteric Python comprised of from-scratch
implementations of popular machine learning algorithms. For more details
and notes on contributing to the project, see the corresponding `GitHub
Repository`_.

.. _GitHub Repository: https://github.com/THargreaves/machine-learning-safari

Installation
------------

You can install the package using Pip:

.. code-block:: console

   $ pip install machine_learning_safari
